# Python-Contact-Dictionary-_CRUD-operation

 ## Project Description

This project is a simple Contact Directory application built using Python.
It allows users to store and manage contact details such as name, phone number, and email.

The application follows CRUD operations:

* Create → Add new contact
* Read → View contact list
* Update → Modify existing contact
* Delete → Remove contact

##  Features

* Add new contacts
* View all contacts
* Update contact details
* Delete contacts
* Menu-driven program

## Technologies Used

* Python
* Lists
* Functions
* User Input

## Learning Outcome

* Understanding CRUD operations
* Working with lists and functions
* Handling user input in Python

 This project is a beginner-friendly implementation of a contact management system.
